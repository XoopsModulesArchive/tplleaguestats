Help: TPL League Stats

Description

Simple application to record and present sport league results/stats

Install/uninstall

No special measures necessary, follow the standard installation process – extract the module folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Operating instructions

To set up this module you need to:

  • Configure your preferences for the module (see ‘Preferences’) and optionally the blocks if you intend to use them (see ‘Blocks’).
  • Add Seasons.
  • Add Teams.
  • Add Matches
